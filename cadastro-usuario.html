Termos de Uso
1. Aceitação dos Termos

Ao utilizar a plataforma "Onde Tem?", você concorda com estes termos de uso. A plataforma destina-se a conectar usuários a estabelecimentos de serviços estéticos.

2. Uso da Plataforma

O usuário compromete-se a fornecer informações verdadeiras e manter seus dados atualizados. É proibido o uso da plataforma para fins ilícitos.

3. Agendamentos

Os agendamentos realizados estão sujeitos à disponibilidade dos estabelecimentos. Cancelamentos devem ser feitos com antecedência mínima de 24 horas.

4. Responsabilidade

A plataforma atua como intermediária entre usuários e estabelecimentos, não se responsabilizando pela qualidade dos serviços prestados.

Política de Privacidade
1. Coleta de Dados

Coletamos dados pessoais necessários para o funcionamento da plataforma, incluindo nome, e-mail, telefone e endereço.

2. Uso dos Dados

Os dados são utilizados para gerenciar sua conta, processar agendamentos e enviar comunicações relevantes.

3. Proteção de Dados

Seguimos as diretrizes da Lei Geral de Proteção de Dados (LGPD) para garantir a segurança das suas informações.

4. Compartilhamento

Seus dados podem ser compartilhados com os estabelecimentos apenas para fins de prestação de serviços agendados.
